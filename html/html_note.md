# 標籤
| 標籤 | 意義 |
| --- | --- |
| \<p\> ... \</p\> | 段落 |
| \<br\> | 換行 |
| \<a href="google.com"\> ... \</a\> | 超連結 |
| \<img src="img_url"\> | 段落 |
| \<!-- ... --\> | 註解 |
| \<pre\> ... \</pre\> | 依照原有的空白、換行顯示 |

# 特殊字元表示法
| 要顯示的字元或符號 | HTML文件中的特殊寫法 |
| --- | --- |
| < | \&lt; |
| > | \&gt; |
| " | \&quot; |
| 空白 | \&nbsp; |
